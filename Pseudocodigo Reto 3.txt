Analisis
Pedir Hemoglobina, Genero 1=hombre, 2=Mujer
Sacar promedio de hemoglobina en mujeres y hombres y mostrando si el promedio es alto, bajo o medio
Mostrar cuantos hombres y mujeres estan por encima del promedio, bajo del promedio e igual al promedio
Pedir cantidad de pacientes

INICIO
Genero, CanPacientes, NumeroH, NumeroM, SuperiorH, IgualH, InferiorH, SuperiorM, IgualM, InferiorM = int
Hemoglobina, SumaHombre, SumaMujer, PromedioH, PromedioM, = float
AlertaH, AlertaM = string

Lea "Digite el numero de pacientes: ";CanPacientes

HemoglobinaH = Vector[CanPacientes]
HemoglobinaM = Vector[CanPacientes]

NumeroH = 0, NumeroM = 0

for i = 0 hasta CanPacientes
    Lea "Digite su hemoglobina en g/dL: "; Hemoglobina
    Lea "Digita su genero 1=Hombre, 2=Mujer: "; Genero

    if Genero == 1
        HemoglobinaH[NumeroH] = Hemoglobina
        NumeroH = NumeroH + 1
        SumaHombre = SumaHombre + Hemoglobina
    elif Genero == 2
        HemoglobinaM[NumeroM] = Hemoglobina
        NumeroM = NumeroM + 1
        SumaMujer = SumaMujer + Hemoglobina
    else
        Escriba "Ingrese un genero valido"
    fin if 
fin for

if NumeroH > 0
    PromedioH = SumaHombre / NumeroH
else
    PromedioH = 0
fin if

if NumeroM > 0
    PromedioM = SumaMujer / NumeroM
else
    PromedioM = 0
fin if

if PromedioH < 12.2 
    AlertaH = "Baja"
elif PromedioH >= 12.2 and PromedioH <= 16.6
    AlertaH = "Normal"
else
    AlertaH = "Alta"
fin if

if PromedioM < 12.6 
    AlertaM = "Baja"
elif PromedioM >= 12,6 and PromedioM <= 15
    AlertaM = "Normal"
else 
    AlertaM = "Alta"
fin if

SuperiorH, IgualH, InferiorH = 0
for i = 0 hasta NumeroH
    if HemoglobinaH[i] > PromedioH
        SuperiorH = SuperiorH + 1
    elif redondear(HemoglobinaH[i], 2) == redondear(PromedioH, 2)
        IgualH = IgualH + 1
    else 
        InferiorH = InferiorH + 1
    fin if
fin for   

SuperiorM, IgualM, InferiorM = 0
for i = 0 hasta NumeroM
    if HemoglobinaM[i] > PromedioM
        SuperiorM = SuperiorM + 1
    elif redondear(HemoglobinaM, 2) == redondear(PromedioM, 2)
        IgualM = IgualM + 1
    else 
        InferiorM = InferiorM + 1
    fin if
fin for 

Escriba redondear(PromedioH, 2), AlertaH;
Escriba redondear(PromedioM, 2), AlertaM;
Escriba SuperiorH, SuperiorM;
Escriba InferiorH, InferiorM;
Escriba IgualH, IgualM;
